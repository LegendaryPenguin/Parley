# Parley — 60-second demo script

The clip that doubles as the submission proof and the community-vote pitch. Shoot it in **live mode** so the "0G does real work" claim is visible on screen.

## Setup (before recording)
1. `.env.local`: `OG_COMPUTE_MODE=live` + a funded Router key. (Storage/Chain can stay mock for a smooth no-wallet take, or set them live to show the wallet beats.)
2. `npm run dev` → `http://localhost:3000`. Use a fresh browser profile so you start at Arrival.
3. Optional: unmute (top-right speaker) so the split-flap clatter + stamp thunk land on camera.
